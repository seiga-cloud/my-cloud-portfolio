# AWS EC2 + nginx Web Server 構築プロジェクト

## 📌 概要
本プロジェクトは、AWS EC2上にWebサーバー（nginx）を構築し、HTMLページを公開するまでの一連の手順を実践した学習プロジェクトです。  
クラウド基礎、Linux操作、Webサーバー構築、Git管理の理解を目的としています。

---

## 🏗 システム構成
- Amazon Web Services（EC2）
- Linux（Amazon Linux）
- nginx Web Server
- HTML（index.html）
- Git / GitHub

---

## 🚀 実装内容
以下の手順で構築を行いました。

1. EC2インスタンスの作成
2. SSH接続設定
3. nginxのインストールと起動
4. HTMLファイルの作成・編集
5. ブラウザでWebページ表示確認
6. Gitによるソース管理
7. GitHubへのリポジトリ公開
