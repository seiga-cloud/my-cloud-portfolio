# Linux 基本コマンド

## ディレクトリ作成

```bash
mkdir ~/aws-web
```

作業用ディレクトリを作成する。

---

## ディレクトリ移動

```bash
cd ~/aws-web
```

作業ディレクトリへ移動する。

---

## ファイル一覧表示

```bash
ls
```

ファイル一覧を表示する。

---

## 詳細表示

```bash
ls -l
```

権限や所有者を含めて表示する。

---

## ファイル編集

```bash
vi index.html
```

viエディタでファイル編集を行う。

---

## vi基本操作

### 入力モード

```bash
i
```

文字入力を開始する。

---

### 保存して終了

```bash
:wq
```

---

### 保存せず終了

```bash
:q!
```

---

## OSアップデート

```bash
sudo dnf update -y
```

パッケージを最新状態へ更新する。
